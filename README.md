# Digital Archives and Methods Portfolio
This repository contains portfolio assignments and the final digital project for the course Digital Archives and Methods, Spring 2026.

## Folders
- `Danish Monarchs`: R Markdown and HTML files for the Danish Monarchs portfolio assignment.
- `GameOfThrones`: Files for the sentiment analysis portfolio assignment.
- `Final Project`: R Markdown file, data and output figures for the final project on official apologies from the Danish state.
